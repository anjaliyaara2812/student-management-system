# Student Management System (Python)

This is a simple command-line based Student Management System developed using Python.

## Features
- Add student details
- View student records
- Update student information
- Delete student records
- Data stored using file handling

## Technologies Used
- Python
- File Handling
- Basic Concepts (Functions, Loops, If-Else)

## How to Run
1. Make sure Python is installed on your system
2. Run the file using:
   python student_management.py

## Project Files
- student_management.py : Main Python program
- students.txt : Stores student data
